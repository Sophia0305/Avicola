Taller de Análisis Exploratorio de Datos con Power BI

Presentación del taller académico

Descripción del Dataset

El dataset analiza la producción avícola por granja, incluyendo variables como producción, ventas, costos y mortalidad.

  • Fecha: Fecha del registro
  • Granja: Nombre (A, B, C, D)
  • Producción (kg)
  • Ventas (kg)
  • Costo Producción ($)
  • Mortalidad (%)

Exploración de Datos

Se cargó el dataset en Power BI, se verificaron tipos de datos y se analizaron valores atípicos en la columna de mortalidad. Se utilizaron filtros, gráficos de dispersión, barras y tablas dinámicas para examinar relaciones entre variables.

Dashboard Interactivo

Se diseñó un panel con gráficos de línea, barras, burbujas e indicadores clave. Incluye segmentaciones por mes, granja y rango de mortalidad.

Ver Dashboard en Power BI Online

Conclusiones

  • Granja B tuvo mayor producción promedio.
  • Existe correlación directa entre producción y ventas.
  • Granja D presentó mayor costo de producción.

El taller permitió aplicar técnicas de análisis exploratorio y visualizar datos de manera interactiva con Power BI.
